Detailed Description

While the remote_service_proc_info_t structure is ready to be used as a list item, list items can only be on one list at a time and we need to track both the service processes per DPU, as well as the service processes to connect to. To avoid any problem, connect_to_service_proc is a container to be used with the list of service procs to connect to that holds a pointer to the service process structure that belongs to a remote_dpu_info_t element.
