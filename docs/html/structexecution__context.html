Detailed Description

execution_context_t is the structure holding all the information related to DPU offloading, both on the hosts and DPUs. The primary goal of the structure is too abstract whether the process is a client or server during bootstrapping and how many clients and servers are used for the deployment of the infrastructure. It recommanded to use the associated macros in order to make the code more abstract and mainly rely on execution contexts.
