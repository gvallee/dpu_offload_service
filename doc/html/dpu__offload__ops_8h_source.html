15  * The registration of an operation only allows the offloading engine to know the definition
16  * of an offloaded operation. It does not instanciate it. Once registered, it can be used to
17  * start and execute many operations. An oepration is defined by three high-level functions that
18  * can be used by developers: init, progress, complete.
19  * To execute an operation, one must get a descriptor. For that, the identifier returned during
20  * the registration must be provided. Once the descriptor is acquired, developers can customized
